MapReduce TSQR

MapReduce TSQR provides several ways to compute the QR decomposition of tall-and-skinny matrices. Since the SVD can be easily computed with QR, the code provides several building blocks for large-scale matrix computations. This is joint work with David Gleich and James Demmel.

The code is available on GitHub: https://github.com/arbenson/mrtsqr

Gamma-ray Imaging Framework (GRIF)

GRIF is a software framework for real-time gamma-ray imaging. The software is designed to allow rapid development of applications for nuclear detection systems. This is work with the BeARING group at UC-Berkeley and LBNL.

The code is available on GitHub: https://github.com/bearing/grif

nacl-mounts

nacl-mounts provides a file system layer for Native Client applications. Native Client i s an open-source project at Google that allows developers to execute native code in the browser. This work was with the Google Native Client team.

The code is available through naclports: http://code.google.com/p/naclports/
